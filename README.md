# Chat App Frontend

Modern real-time chat application frontend built with Nuxt 4.

## Features

- Authentication (Login / Register)
- JWT / Sanctum Integration
- Real-Time Messaging
- Room Management
- State Management with Pinia
- Responsive UI
- Tailwind CSS
- Laravel Reverb Integration

## Tech Stack

- Nuxt 4
- Vue 3
- TypeScript
- Pinia
- Tailwind CSS
- Laravel Echo
- Pusher JS
- Axios / $fetch
